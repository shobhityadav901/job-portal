# 🚀 Job Portal - Flask Web Application

Ek full-stack Job Portal system jise **Python** aur **Flask** ka use karke banaya gaya hai. Isme users jobs dekh sakte hain, apply kar sakte hain aur admin jobs post kar sakta hai.

## ✨ Features
* **User Registration & Login:** Naya account banayein aur login karein.
* **Job Listings:** Saari available jobs ek hi jagah dekhein.
* **Job Application:** Direct portal se jobs ke liye apply karein.
* **Post Jobs:** Admin nayi job openings add kar sakta hai.
* **Database:** SQLite ka use karke data store kiya gaya hai.

## 🛠️ Tech Stack
* **Backend:** Python 3.x, Flask
* **Frontend:** HTML5, CSS3, JavaScript
* **Database:** SQLite3
* **Hosting Support:** Render / PythonAnywhere

## 📂 Project Structure
job-portal/
├── app.py              # Main Flask application logic

    ├── apply.html
    └── post_job.html

🚀 How to Run Locally
Repository ko Clone karein:

Bash
git clone [https://github.com/shobhityadav901/job-portal.git](https://github.com/shobhityadav901/job-portal.git)
cd job-portal
Dependencies Install karein:

Bash
pip install -r requirements.txt
App Start karein:

Bash
python app.py
Browser mein open karein: http://127.0.0.1:5000
